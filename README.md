TGMC quotesdb with additional functionality, forked from https://github.com/psykzz/cogs

What's changed:
 - manage_messages required for deleting quotes instead of being a full discord admin
 - other minor permission changes
 - added a function to remove all quotes in a trigger
 - added a function to remove all quotes added by a user
 - added a ban function to prevent a user from adding new quotes
 - added a cooldown on invoking quotes outside of designated spam channels (30s)
 - also fixes authors not displaying while using the qid command (possibly my fault)
